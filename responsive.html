Stay Flexy

For the most part, using responsive values such as max or min took care of the responsively of the layout between large and medium desktop sizes. however the design did break around 650px so a single media query was used. The function of this media query was MAINLY to switch from display:grid layouts to display: flex layouts. As when a layout is adapted for mobile it is primarily a single column layout, something flex is very good at.
